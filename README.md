PhishGuard — учебный проект: браузерное расширение (Manifest V3) + сервер (FastAPI) для проактивной проверки ссылок и текста на признаки фишинга.
Что делает
- Проверяет URL и текст на признаки фишинга.
- Показывает пользователю итоговый риск и причины срабатываний.

Структура репозитория
- `backend/` — серверная часть (FastAPI)
- `extension/` — браузерное расширение (Chrome/Edge)
